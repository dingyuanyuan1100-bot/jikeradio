# Mineradio v1.1.1

界面展示名：`Jike radio`

Mineradio 是一款 Windows 端可视化动态壁纸音乐播放器，主打多平台音乐账号互通、桌面歌词和视频动态壁纸联动。它不只是播放器，也是一座视觉舞台。

![启动欢迎页](./images/welcome.png)

## 本版亮点

- 支持网易云音乐、QQ 音乐、酷狗音乐多平台登录与音乐库同步
- 支持本地导入视频/图片作为动态壁纸素材
- 支持桌面歌词、粒子特效、歌单架、沉浸式播放界面联动
- 支持 DIY 玩家模式，提供更完整的视觉与播放控制
- 支持系统托盘、全局快捷键、单实例唤起、更新面板等桌面能力

![多平台登录](./images/login.png)

## 主要体验

- 启动后可从欢迎页进入主界面
- 无需登录也可以先搜索歌曲试听
- 登录后可同步歌单、红心、播客等平台数据
- 壁纸、歌词、粒子与音乐节拍联动
- 可在壁纸素材库中切换内置素材，或从本地导入新素材

![壁纸素材库](./images/wallpaper-library.png)

## 成品效果

完整播放桌面由四部分组成：

1. 底层动态壁纸
2. 居中歌词与粒子特效
3. 右侧悬浮歌单架
4. 底部全局播放控制台

![完整播放桌面](./images/player-desktop.png)
