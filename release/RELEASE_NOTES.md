## 软件简介

Jikeradio 是一款 Windows 端可视化动态壁纸音乐播放器，主打多平台音乐账号互通与视频动态壁纸联动。软件本体名称为 `Jikeradio`，界面展示名为 `Jike radio`。

它不只是一个听歌工具，也是一座视觉舞台。用户既可以把它当作常规播放器使用，也可以把它作为桌面动态壁纸、歌词展示和音乐氛围渲染工具来使用。

![图4 启动欢迎页](./images/welcome.png)

![图1 多平台登录弹窗](./images/login.png)

1. 平台切换栏：顶部支持网易云音乐、QQ 音乐、酷狗音乐、汽水音乐四大主流音源切换，图中当前选中的是酷狗音乐。

3. 登录能力：登录后可同步歌单、红心、播客等个人音乐数据，壁纸、歌词、粒子特效会随音乐联动。
4. 登录方式：支持对应平台 App 扫码登录。图中二维码已完成扫码，状态提示为“已扫码，请在手机确认”。
5. 底部功能按钮：
   - `取消`：关闭登录弹窗。

   - `刷新二维码`：当二维码失效时重新生成。
6. 背景底层：弹窗后方仍可看到主界面预览，包括时间、标语、壁纸设置和播放器控制台入口。

### 图 2：壁纸素材库

![图2 壁纸素材库](./images/wallpaper-library.png)

1. 窗口标题：`WALLPAPER LIBRARY 选择一张开场壁纸`，这是软件的核心特色功能区之一。
2. 素材规则：支持 16:9 图片与视频壁纸，卡片左上角会显示 `VIDEO` 标识，用于区分视频素材。
3. 内置素材示例：包含动漫风景列车、二次元霓虹少女、水下动漫人像、黑白漫画人像等预制 MP4 动态壁纸。
4. 操作功能：
   - `上一页 / 下一页`：切换素材分页，图中素材库共 3 页。
   - `从本地导入`：可载入电脑中的本地 MP4 视频作为桌面动态壁纸。
   - `关闭`：退出素材库窗口并返回主播放界面。
5. 路径显示：顶部会显示本地素材存放文件夹路径，方便用户自行添加或维护壁纸资源。

### 图 3：完整播放桌面成品

![图3 完整播放桌面成品](./images/player-desktop.png)

   - 播放进度条可显示歌曲总时长与当前播放位置。

## 软件整体核心特点

1. 双功能融合：普通音乐播放器与动态桌面壁纸二合一，无需同时开启 Wallpaper Engine 和音乐软件。
2. 多平台音源互通：支持四大音乐平台账号登录，并同步个人收藏、歌单等数据。
3. 高自定义视觉：支持本地导入任意视频作为壁纸，歌词、粒子、界面组件均可 DIY。
4. 轻量化低占用：相比专业壁纸软件，性能消耗更低，更适合日常听歌与桌面美化同时进行。

## 已实现能力补充

- 沉浸式视觉播放器，支持封面动画、歌词、粒子、镜头运动和舞台化效果。
- DIY 玩家模式，提供更完整的视觉控制、歌词控制、播放控制与高级调节能力。
- 多平台登录已接入网易云音乐、QQ 音乐、酷狗音乐；汽水音乐界面入口已预留。
- 支持桌面歌词悬浮窗，可锁定、拖动、穿透，并保持实时歌词同步。
- 支持壁纸模式，可将播放器视觉内容附着到 Windows 桌面层。
- 支持全局快捷键、系统托盘、单实例唤起、更新面板、JSON 导入导出等桌面能力。
